# 機能要件

## プロジェクト概要

Azure Functions (Python) を使用した計算APIの実装。ブラウザからアクセス可能な掛け算と割り算の2つのAPIエンドポイントを提供する。

## API仕様

### 1. 掛け算API

**エンドポイント:** `/api/multiply`

**機能:** 2つの数値パラメータの積を計算し、結果をHTML形式で返す

**HTTPメソッド:**
- GET
- POST

**リクエストパラメータ:**
| パラメータ名 | 型 | 必須 | 説明 |
|------------|------|------|------|
| A | 小数 | ○ | 掛けられる数（正の小数） |
| B | 小数 | ○ | 掛ける数（正の小数） |

**レスポンス形式:**
- Content-Type: `text/html; charset=utf-8`
- スタイリングされたHTML形式で計算結果を表示

**正常系の動作:**
- パラメータAとBを受け取り、A × B の結果を計算
- スタイリングされたHTMLページで結果を表示

---

### 2. 割り算API

**エンドポイント:** `/api/divide`

**機能:** 2つの数値パラメータの商を計算し、結果をHTML形式で返す

**HTTPメソッド:**
- GET
- POST

**リクエストパラメータ:**
| パラメータ名 | 型 | 必須 | 説明 |
|------------|------|------|------|
| A | 小数 | ○ | 割られる数（正の小数） |
| B | 小数 | ○ | 割る数（正の小数） |

**レスポンス形式:**
- Content-Type: `text/html; charset=utf-8`
- スタイリングされたHTML形式で計算結果を表示

**正常系の動作:**
- パラメータAとBを受け取り、A ÷ B の結果を計算
- スタイリングされたHTMLページで結果を表示

---

## 入力値の制約

### 有効な入力値
- **データ型:** 小数（浮動小数点数）
- **値の範囲:** 正の数（0より大きい値）
- **範囲制限:** なし（上限・下限は設定しない）

### 無効な入力値
- 負の数
- ゼロ
- 数値以外の文字列
- パラメータの未指定

---

## エラーハンドリング

### エラーケースとメッセージ

#### 1. パラメータ未指定
**条件:** パラメータAまたはBが指定されていない場合

**動作:** エラーメッセージをHTML形式で表示

**エラーメッセージ例:**
```
パラメータAとBを指定してください
```

---

#### 2. 数値以外の入力
**条件:** パラメータAまたはBに数値として解釈できない値が指定された場合

**動作:** エラーメッセージをHTML形式で表示

**エラーメッセージ例:**
```
AとBは数値で指定してください
```

---

#### 3. ゼロ除算（割り算APIのみ）
**条件:** 割り算APIでパラメータBに0が指定された場合

**動作:** エラーメッセージをHTML形式で表示

**エラーメッセージ例:**
```
0で割ることはできません
```

---

#### 4. 負の数または0の入力
**条件:** パラメータAまたはBに0以下の値が指定された場合

**動作:** エラーメッセージをHTML形式で表示

**エラーメッセージ例:**
```
AとBは正の数を指定してください
```

---

## HTMLレスポンスの要件

### 正常時のレスポンス
- 計算結果を見やすく表示
- スタイリングされた見た目（CSS適用）
- 計算式と結果を明示的に表示

### エラー時のレスポンス
- エラーメッセージを分かりやすく表示
- スタイリングされた見た目（CSS適用）
- エラーの種類に応じた適切なメッセージ

### スタイリング要件
- ブラウザで見やすいデザイン
- レスポンシブ対応不要（基本的なスタイリングのみ）
- 視認性を重視したシンプルなデザイン
